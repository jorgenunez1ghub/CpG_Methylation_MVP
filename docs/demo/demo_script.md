# Demo Script (5–7 minutes)

## 1) Scope and disclaimer (30–45 sec)
- This is an educational CpG methylation MVP, not medical advice.
- Workflow 01 scope: upload → validate/normalize → QC → panel coverage → cautious interpretation → local cited context.

## 2) Launch and orient (45–60 sec)
- Open the app and show duplicate policy selector.
- Call out upload size limit and session-scoped artifacts.

## 3) Upload walkthrough (60–90 sec)
- Upload a safe sample file from `data/sample/`.
- Show processing report details (row counts, parse strategy, warnings, duplicate handling).

## 4) QC and explanation (60–90 sec)
- Review QC metrics and beta histogram.
- Review “What this QC summary means” sections:
  - observed data
  - possible meaning
  - limitations
  - recommended next steps

## 5) Interpretation and context (60–90 sec)
- Show curated panel coverage and structured interpretation summary.
- Emphasize limitations and non-clinical boundaries.
- Show local cited context table.

## 6) Export artifacts (30–45 sec)
- Download normalized CSV and report artifacts.
- Mention duplicate review/audit files when present.

## 7) Fallback plan (30 sec)
- If hosted app fails: run locally and continue using the same sample walkthrough.
- If upload fails: switch to known-good sample and explain file contract expectations.
